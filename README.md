# Waste2Value
Hackathon Project
Team Name: BITstack

Problem Statement
Create a technology driven solution to tackle pollution, biodiversity loss, and resources overuse in marine ecosystems.

Solution
We have made a website callled Waste2Value that focuses on an unique approach to trash removal(water polllution) from the oceans.Any interested individual can clean,buy or sell the ocean waste and earn special tokens that be exchanged for exclusive merchandise and other products.We also have a map feature where a seller can pinpoint the location of the waste so its easier for the buyer to find and claim it.In this way,our solution deviates from traditional trash removal methods and focuses on fun and engaging techniques that will pique the interest of the customer while also staying true to its original goal of tackling pollution.

Tech Stack
Frontend:HTML5,CSS3,Leaflet.js
Backend: none
Database:Local storage
Mapping:Geolocation API
How to Run
Run it on VS Code live server
